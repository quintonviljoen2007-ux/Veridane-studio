# Veridane Studio Website

A complete black-and-white services website for selling beginner-friendly web design services.

## Files

- `index.html` — main website structure
- `style.css` — full responsive styling
- `script.js` — interactive UI, pricing toggle, quote estimator, FAQ, form behaviour
- `assets/logo.svg` — corporate wordmark logo
- `assets/favicon.svg` — browser icon / compact logo

## Before publishing

1. Open `script.js`.
2. Replace:

```js
const businessEmail = "your.email@example.com";
```

with your real email address.

3. Open `index.html` and replace this placeholder URL inside the schema block:

```json
"url": "https://your-domain-here.com"
```

with your live website URL.

4. Replace placeholder work examples with real projects when you have them.

## Contact form note

The current form uses a `mailto:` fallback. That means it opens the visitor's email app with a pre-filled message. For a more professional setup, connect the form to Formspree, EmailJS, Netlify Forms, or FormSubmit.
